# AtlasCloud Network Orchestrator - Branding & Style Guide

This document captures the visual identity and styling guidelines for the **AtlasCloud Network Orchestrator** platform.

## 1. Visual Identity & Brand
- **Product Name**: AtlasCloud Network Orchestrator
- **Tagline**: Real-Time Multi-Vendor SDN Orchestration, Telemetry, and Cryptographic Configuration Lifecycles
- **Target Palette**: Slate, Deep Violet/Indigo, and Cyan accents (clean, premium dark-mode interface, completely isolated from legacy NacTrack teals/greens).

## 2. Color Palette
- **Background Dark**: `#07080e` (Solid backdrop)
- **Glass Panel Fill**: `rgba(15, 16, 28, 0.75)` with `backdrop-filter: blur(25px)`
- **Card Background**: `rgba(26, 28, 48, 0.4)`
- **Primary Accent / Borders**: `#8b5cf6` (Deep Violet)
- **Secondary Accent**: `#06b6d4` (Teal-Cyan)
- **Compliance Compliant (Active/Success)**: `#10b981` (Green)
- **Compliance Drift (Warning)**: `#f59e0b` (Amber)
- **Error / Rejected Findings**: `#ef4444` (Red)

## 3. Typography
- **Headings & Titles**: `'Outfit', sans-serif` (Premium, modern geometric)
- **Body & Tabular Data**: `'Inter', sans-serif` (High readability)
